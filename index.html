🌐 Live Access on Multiple Systems

Since this is a client-side web application, you can easily open and run it anywhere:

  • Offline/Local Copy: Copy this entire folder to a USB drive or send it via email. Simply double-click on index.html on any system to open the dashboard.
  • Data Syncing: Use the Export Data button to save your changes, copy the JSON file to your other computer, and click Import Data to upload it.
  • Free Hosting: You can host this folder on free static hosts like GitHub Pages, Vercel, Netlify, or Surge to access it via a live URL on any mobile, tablet, or laptop worldwide!
